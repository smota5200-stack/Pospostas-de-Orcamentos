# Proposify - Sistema de Gestão de Orçamentos 🚀

Sistema completo para criação, gestão e acompanhamento de orçamentos focado em agilidade e design premium.

## ✨ Funcionalidades

- **Dashboard**: Visão geral de vendas, orçamentos e clientes.
- **Gerador de Propostas**: Criação de PDFs profissionais com design customizado.
- **Lixeira (Soft Delete)**: Exclusão segura com retenção de 30 dias.
- **Design Responsivo**: Totalmente funcional em Celulares, Tablets e Desktops.
- **Integração Supabase**: Banco de dados na nuvem para acesso de qualquer lugar.

## 🛠️ Tecnologias

- **Frontend**: React, Tailwind CSS, Radix UI.
- **Backend**: Node.js, Express.
- **Banco de Dados**: Supabase (PostgreSQL).
- **Controle de Versão**: Git.

## 🚀 Como Rodar Localmente

1. **Instalar dependências**:
   ```bash
   npm install
   ```

2. **Configurar variáveis de ambiente**:
   Crie um arquivo `.env` na raiz com:
   ```env
   VITE_SUPABASE_URL=https://wvncdusvpfbdsnclynxl.supabase.co
   VITE_SUPABASE_ANON_KEY=sua_chave_anon_do_supabase
   ```

3. **Rodar em modo desenvolvimento**:
   ```bash
   npm run dev
   ```

## 🌐 Deploy Online

Este projeto está configurado para deploy automático no **Render.com** através do arquivo `render.yaml`.
Para publicar, basta empurrar o código para um repositório no GitHub e conectar ao Render.
